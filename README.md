# Groupware Frontend

## Domain

https://groupwares.io

## Deployments

AWS ECS

## Stack

- Next js (with Typescript)
- Redux
- Tailwind css

## Install & Start

```bash
pnpm install

pnpm run start

# dev
pnpm run dev
```

## Structure

```bash
└── app
    ├── (route)
    ├── components
    ├── constant
    ├── module
    │   ├── hooks
    │   └── utils
    ├── providers
    ├── store
    └── types
```
